# Python Libraries Repository

- A collection of structured practice notebooks and scripts covering essential Python libraries used in Data Science, Machine Learning, and AI.

- This repo demonstrates hands-on mastery of the core tools required for real world data work from numerical computing to data analysis and visualization.

## This Repository Covers

### 1. NumPy
- Array creation & manipulation
- Indexing, slicing, boolean masking
- Broadcasting
- Vectorized operations
- Mathematical/statistical functions
- Reshaping, stacking, splitting
- Performance intuition
### 2. Pandas
- Series & DataFrame operations
- Data cleaning
- Missing value handling
- Feature engineering basics
- Filtering, grouping, merging
- Time series operations
- Case studies
### 3. Matplotlib
- Line, bar, scatter, histogram plots
- Subplots
- Styles, annotations
- Customizations
### 4. Seaborn
- Relational, categorical & distribution plots
- Pairplot, heatmap
- Statistical visualizations

## Purpose of This Repo

**This repository is built to:**

- Practice key DS/ML libraries deeply
- Build intuition through examples
- Create a reference you can revisit anytime
- Show employers real, continuous progress
- Support future ML/DL/MLOps projects

## Tech Stack
